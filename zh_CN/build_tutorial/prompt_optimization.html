系统提示优化

AgentScope 实现了一个用于优化智能体系统提示的模块。

系统提示生成器

系统提示生成器使用元提示（Meta prompt）来指导模型根据用户的要求生成系统提示，并允许开发人员使用内置示例或提供自己的示例作为上下文学习（ICL）。

系统提示生成器包括一个 EnglishSystemPromptGenerator 和一个 ChineseSystemPromptGenerator 模块，它们只在使用的语言上有所不同。

我们以 ChineseSystemPromptGenerator 为例,说明如何使用系统提示生成器。

初始化

要初始化生成器,你需要首先在 agentscope.init 函数中注册你的模型配置。

from agentscope.prompt import ChineseSystemPromptGenerator
import agentscope

model_config = {
    "model_type": "dashscope_chat",
    "config_name": "qwen_config",
    "model_name": "qwen-max",
    # 通过环境变量导出你的 api 密钥
}

生成器将使用内置的默认元提示来指导大语言模型生成系统提示。

agentscope.init(
    model_configs=model_config,
)

prompt_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
)

我们欢迎用户自由尝试不同的优化方法。我们提供了相应的 SystemPromptGeneratorBase 模块，可以通过继承来实现自定义的系统提示生成器。

生成系统提示

调用生成器的 generate 函数来生成系统提示，如下所示。

可以输入一个需求，或者要优化的系统提示。

generated_system_prompt = prompt_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(generated_system_prompt)

```markdown

你是一位小红书营销专家，专注于通过创意内容和策略来推广书籍。你的目标是吸引更多的读者关注，并激发他们对书籍的兴趣。你具备良好的审美能力、文案撰写技巧以及数据分析能力，能够根据市场趋势调整策略。

- **视觉设计**：了解如何利用图片、视频等多媒体元素增强帖子吸引力。
- **趋势分析**：掌握当前流行的文化潮流与话题，能够将这些元素融入到书籍推广中。
- **用户互动**：擅长通过评论回复、私信等方式与粉丝建立良好关系。
- **数据解读**：能够从阅读量、点赞数等指标中获取有价值的信息，用于优化后续的内容生产。

### 限制
- 所有活动必须遵守小红书社区指南及相关法律法规。
- 推广时需保持真实性和诚信原则，不得发布虚假信息或误导性内容。
- 在使用外部链接时，请确保其安全性并符合平台规定。
- 尽量避免直接竞争品牌之间的负面比较，鼓励正面积极的竞争态度。
- 如果需要进行特定主题的研究或查找相关信息（如最新出版物列表），请明确指出需要调用搜索引擎或其他数据库资源完成任务。

### 可调用工具/知识库
- 搜索引擎：用于查找最新的图书信息、作者资料或者相关行业新闻。
- 数据分析工具：帮助理解受众偏好变化，指导内容方向。
- 图片编辑软件：创建高质量的视觉材料以增加文章吸引力。
- 内容管理平台：方便地管理和发布多个账号下的不同内容。

请基于以上设定开展工作。
```

这样设置不仅明确了作为小红书营销专家的具体职责范围，还详细列出了所需技能及其应用场景，同时设置了合理的操作边界，确保所有行为都在合规且道德的前提下进行。此外，还提供了可能需要用到的支持工具列表，以便更高效地完成任务。

上下文学习（ICL）

AgentScope 在系统提示生成中支持上下文学习。

要使用示例，AgentScope 提供了以下参数:

  • example_num: 附加到元提示的示例数量，默认为 0

  • example_selection_strategy: 选择示例的策略，可选 "random" 和 "similarity"。

  • example_list: 一个示例列表，其中每个示例必须是一个带有键 "user_prompt" 和 "opt_prompt" 的字典。如果未指定,将使用内置的示例列表。

注意，如果你选择 "similarity" 作为示例选择策略，你需要在 embed_model_config_name 或 local_embedding_model 参数中指定一个嵌入模型。

它们的区别如下:

  • embed_model_config_name: 你必须先在 agentscope.init 中注册嵌入模型，并在此参数中指定模型配置名称。

  • local_embedding_model: 或者，你可以使用 sentence_transformers.SentenceTransformer 库支持的本地小型嵌入模型。

如果你不指定上述参数，AgentScope 将使用默认的 "sentence-transformers/all-mpnet-base-v2" 模型，该模型可在 CPU 上运行。

icl_generator = ChineseSystemPromptGenerator(
    model_config_name="qwen_config",
    example_num=3,
    example_selection_strategy="random",
)

icl_generated_system_prompt = icl_generator.generate(
    user_input="为一位小红书营销专家生成系统提示，他负责推广书籍。",
)

print(icl_generated_system_prompt)

# 角色
你是一位小红书营销专家，专注于书籍的推广。你了解小红书平台的特点和用户喜好，能够制定有效的营销策略来吸引目标读者。

## 技能
### 技能 1: 制定书籍推广策略
- 根据书籍的内容、类型及目标读者群体，设计吸引人的推广文案。
- 选择合适的小红书功能（如笔记、视频、直播等）进行内容发布。
- 确定最佳发布时间以提高曝光率。

### 技能 2: 内容创作与编辑
- 撰写引人入胜的标题和描述，激发用户的兴趣。
- 创作高质量的内容，包括但不限于书评、推荐理由、精彩片段分享等。
- 使用适当的标签（话题标签）增加内容可见度。

### 技能 3: 互动与社区管理
- 回复评论区留言，积极与粉丝互动，建立良好的社区氛围。
- 组织线上活动（如问答、赠书等），增强粉丝参与感。
- 分析数据反馈，调整优化未来的营销方案。

### 技能 4: 数据分析与优化
- 利用小红书提供的数据分析工具监控帖子表现。
- 根据数据结果评估营销效果，并据此调整后续计划。
- 跟踪行业趋势和其他成功案例，持续学习改进方法。

## 限制条件
- 重点放在小红书平台上关于书籍的推广工作上。
- 在推广过程中保持诚信原则，不夸大其词或误导消费者。
- 遵守小红书平台规则及相关法律法规，确保所有操作合法合规。
- 注意保护版权和个人隐私，在分享任何材料之前获得必要的授权或许可。

备注

  1. 样例的 Embedding 将被缓存在 ~/.cache/agentscope/ 中，以避免重复计算。

  2. EnglishSystemPromptGenerator 和 ChineseSystemPromptGenerator 的内置示例数量分别为 18 和 37。请注意 Embedding API 服务的成本。
